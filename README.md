# AvdCntrlr

Ruby code to dynamically update android virtual device instance from gpsd

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'avd_cntrlr'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install avd_cntrlr
